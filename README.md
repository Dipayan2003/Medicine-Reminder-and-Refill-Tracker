# Medicine Reminder and Refill Tracker (Backend)

A Spring Boot backend application to manage medicines, reminders, and refills for logged-in users.

## Features
- User login using Spring Security (session-based)
- Medicine CRUD operations per logged-in user
- No JWT / No password encryption (learning-focused)
- REST APIs tested with Postman
- MySQL database integration
- Clean layered architecture

## Tech Stack
- Java
- Spring Boot
- Spring Security
- Spring Data JPA
- MySQL
- Lombok

## API Modules
- User
- Medicine
- Reminder (upcoming)

## How to Run
1. Clone the repository
2. Configure `application.properties`
3. Run the Spring Boot application
